HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Removing HTML elements will make the page hard to read

CSS

  • A margin indicates how much space we want around the outside of an element
  • A pading indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git command must proceed every Git statement
  • git add -A: adds modifications to the current working branch to the staging area
  • git commmit -m "commit message": Commit changes to the branch that is being worked on
  • ALWAYS check that your local branch is in sync with the base branch in github "git pull origin main"
  • git push origin "branch name": This will push the changes over to GitHub

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
